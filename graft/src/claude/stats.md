# src/claude/stats.ts

This module provides functions to read wiring data from a JSON file and compute statistics about graph nodes and edges, facilitating analysis of project structure.

- readWiring · function · L6-L10 — Loads and parses the wiring configuration from a JSON file, returning the graph structure or null if an error occurs.
- computeStats · function · L12-L25 — Calculates and returns statistics about the nodes and edges in a graph, including counts and languages used.
